What is SignSpeak?

?? About Our Project

SignSpeak is a web-based learning platform we created as part of our capstone project. We wanted to build something that could help people learn basic American Sign Language (ASL) using simple web technologies and tools.

This is a student project created for educational purposes. While we hope it's helpful for basic ASL learning, it's important to remember that we're still learning and this represents our current skill level rather than a professional product.

??? What We Built

Our platform includes several simple tools to help beginners get started with ASL:

Basic Learning Content

Simple lessons covering ASL alphabet, numbers, colors, and common words to get started with sign language.

Camera Practice Tool

A basic webcam feature that attempts to recognize hand signs using available web technologies.

Visual Learning

Video demonstrations and images to help learners see how signs are formed.

Text Helper

A simple tool to convert text into sign language demonstrations.

?? How We Built It

For our capstone project, we used Google's MediaPipe technology to handle the hand tracking features. We built this as a web application using HTML, CSS, and JavaScript to make it accessible to anyone with a web browser.

Technologies Used:

  • HTML, CSS, and JavaScript for the web interface
  • MediaPipe for hand gesture recognition
  • Video demonstrations for visual learning
  • Responsive design for different devices
