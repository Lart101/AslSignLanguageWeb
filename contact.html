Is SignSpeak really free to use?

+

Yes! SignSpeak is completely free as it was developed as an educational capstone project. Our goal is to make ASL learning accessible to everyone, regardless of financial circumstances.

What devices and browsers are supported?

+

SignSpeak works on most modern browsers (Chrome, Firefox, Safari, Edge) and devices with camera access for the webcam features. Mobile devices are supported for most features, though webcam functionality works best on desktop/laptop computers.

How accurate is the sign recognition?

+

Our basic recognition system uses MediaPipe technology and works reasonably well for simple signs, but it's still a learning project and may not always be accurate. We recommend using it as a practice tool alongside proper ASL instruction from qualified teachers.

Can I use SignSpeak for commercial purposes?

+

This is a student capstone project created for educational purposes. You can check our GitHub repository for licensing details. Since this is a learning project, we encourage educational use, but please remember this isn't a professional product.

Do you offer offline functionality?

+

Currently, SignSpeak needs an internet connection for the camera recognition features since they use web-based tools. Most of the learning content like videos and images work offline once loaded, but the interactive camera features require being online.

How can I contribute to the project?

+

Thanks for your interest! You can help by reporting bugs or suggesting improvements through our GitHub repository. As this is a student project, we also appreciate feedback from anyone in the ASL or deaf community who can help us learn and improve.

Is this a professional ASL learning platform?

+

No, SignSpeak is a student capstone project created for learning purposes. While we hope it's helpful for basic ASL practice, it shouldn't replace proper instruction from qualified ASL teachers or formal courses.

What if I find errors or have suggestions?

+

Please let us know! You can email us or create an issue on our GitHub repository. As students, we're always learning and appreciate constructive feedback to help us improve the project.
